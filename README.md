# requinOpx 2.0 CRM

A next-generation, high-performance CRM and Enterprise Operations platform designed for high-growth teams. Built with speed, intelligence, and security at its core.

![requinOpx 2.0 CRM Preview](https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=format&fit=crop&q=80&w=2070)

## 🚀 Overview

requinOpx 2.0 CRM is a comprehensive suite for managing the entire business lifecycle. From lead acquisition to project delivery and billing, it provides a unified interface with AI-driven insights to help teams close deals faster and build lasting customer relationships.

## ✨ Key Features

- **📊 Intelligent Dashboard**: Real-time revenue tracking, conversion analytics, and AI-generated business insights.
- **🎯 Lead Management**: Advanced lead scoring and tracking to prioritize high-probability conversions.
- **🤝 Client Relations**: Centralized database for managing client profiles, history, and communication.
- **📈 Sales Pipeline**: Visual Kanban-style pipeline for managing deals across multiple stages.
- **🛠️ Support Desk**: Omnichannel ticketing system with priority-based sorting and SLA tracking.
- **👥 HR Operations**: Employee management, role-based access, and organizational oversight.
- **📂 Project Management**: Task tracking, deadline management, and collaborative project workflows.
- **💳 Billing & Invoicing**: Automated invoice generation, GST calculations, and payment status tracking.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.

## 🛠️ Tech Stack

- **Frontend**: React 18+, TypeScript, Vite
- **Styling**: Tailwind CSS (Utility-first CSS)
- **Animations**: Framer Motion (`motion/react`)
- **Icons**: Lucide React
- **Charts**: Recharts (D3-based visualization)
- **Notifications**: Sonner

## 🏗️ Project Structure

```text
src/
├── components/       # UI components and Module implementations
│   ├── LandingPage.tsx # High-impact landing page
│   └── Modules.tsx     # Core CRM module components
├── data/             # Mock data and initial state
├── lib/              # Utility functions (cn, formatters)
├── types/            # TypeScript interfaces and enums
└── App.tsx           # Main application entry and state management
```

## 🚦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 🔒 Security

- **Role-Based Access Control (RBAC)**: Supports Admin, Sales, HR, and Support roles with filtered module access.
- **Data Privacy**: Built-in patterns for secure data handling and PII protection.
